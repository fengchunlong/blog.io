列表更删改查

增加

name = [“张三”,”李四”,”王五”]

  • name.append(‘赵六’) # [“张三”,”李四”,”王五”,”赵六”]
  • name.insert(‘位置’,’插入的值’’)
  • name.extend(name2); #添加已有列表，组成新列表
  • a+=a的形式

删除

  • name.pop(); # ‘张三’, ‘李四’]
  • name.remove(‘王五’); # 只删除第一个
  • name[0:2] #切片
  • del name[下标]

修改

  • 赋值修改

查询

-in # 返回布尔值

  • not in

字典更删改查

student = {‘name’:’张三’,’性别’:‘男’}

新增

student[‘age’] = 18

删除

del student[‘age’]

修改

student[“name”] = “李四”

查找

student.get(‘name’) # 获取值

元组

元组与列表类似，内部值不能更改，有点类似PHP中的常量。
例如，列表是一个普通文件，元素就是只读文件。
元组只能查看。

总结

  • 可变类型： 列表，字典。不能做为键名，因为值可变，哈希算法无法找到。
  • 不可变类型：其他
